# 🟢 ECOBICI CDMX: Dashboard de Disponibilidad en Tiempo Real

Este proyecto es un tablero interactivo desarrollado con **Python**, **Streamlit** y **Plotly** que permite visualizar la ubicación y el estado actual (bicicletas y puertos disponibles) de las cicloestaciones del sistema ECOBICI en la Ciudad de México.



## 🚀 Características
* **Mapa Interactivo:** Visualización geográfica de todas las estaciones.
* **Datos en Tiempo Real:** Conexión directa a los feeds GBFS de ECOBICI.
* **Métricas Clave:** Indicadores dinámicos de total de bicis y puertos libres.
* **Filtros Inteligentes:** Búsqueda por nombre de estación o disponibilidad mínima.

## 🛠️ Stack Tecnológico
* **Lenguaje:** Python 3.x
* **Librerías:**
    * `Streamlit`: Para la interfaz de usuario web.
    * `Plotly`: Para los mapas y gráficos interactivos.
    * `Pandas`: Para la manipulación y cruce de datos (ETL).
    * `Requests`: Para el consumo de la API de ECOBICI.
